## This a project for testing our git knowledge 🔥🤝🏽🦾

## How to get started

- Fork the repo
- Clone the repo
- Create and checkout to a new branch with the following format <your-name>:<branch-name>
- Write some code 👩🏽‍💻👩🏽‍💻
- Commit your code
- Make a pull request

### Do not commit to master branch ⚠
